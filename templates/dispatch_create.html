Creating Dispatches as Supplier
  • Select an intermediate user who will receive the components
  • Add one or more items to the dispatch
  • For each item, select the component and specify quantities
  • FLC Quantity represents the number of FLC units
  • Component Quantity represents the actual component count
  • You can add remarks for each item if needed
  • Note: Total FLC quantity cannot exceed available inventory ({{ available_flcs }})

Creating Dispatches as Intermediate
  • Select from available dispatches you received from suppliers
  • For each source dispatch, add end users who will receive components
  • Specify quantities for each end user
  • The total quantity cannot exceed available stock from the source
  • You can distribute one source dispatch to multiple end users
